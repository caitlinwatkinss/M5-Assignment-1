# SCOPE Worksheet Draft

## S — Set the objective
- Produce a concise, decision-useful HOA financial report for board use using two source packets.
- Prioritize year-end reliability (12/31/2023) while keeping June 2023 as the comparison baseline.

## C — Context and data
- Primary dataset: **Financials Packet 2.pdf** (year-end balance sheet, December operating statement, comparative statement, bank reconciliation, detailed GL).
- Secondary dataset: **Financials Packet 1.pdf** (midyear balance sheet, June operating statement, comparative statement, bank reconciliation).
- Excluded low-value pages: blank deposit/check pages, disclosure pages, filler support pages with no decision-useful financial figures.

## O — Output and constraints
- Outputs produced:
  - page_map.md
  - extracted_financials.md
  - condensed_balance_sheet.csv
  - condensed_operating_results.csv
  - verification_checklist.md
  - report_storypoints.md
  - chart_recommendations.md
  - methods_note_draft.md
  - scope_worksheet_draft.md
- Constraints followed:
  - Used only traceable values from source pages.
  - Marked uncertain/partially visible items as REVIEW NEEDED.
  - Focused on concise board-report usefulness, not packet completeness.

## P — Proof and checks
- Balance sheet cash tied to bank reconciliations for both June and December.
- December monthly expense total tied arithmetically and supported with GL-level entries for key accounts.
- Flagged returned payment and limited visibility areas for manual follow-up.

## E — Evaluate and extend
- Immediate extension: add a one-page dashboard with 4 charts (cash/AR trend, monthly margin, YTD actual vs budget, top over-budget categories).
- If more time: add invoice-level audit trail for all December checks and outstanding-item rollforward from bank recon pages.
- For board follow-up: monitor irrigation/water spending controls and collection process around returned/late payments.
